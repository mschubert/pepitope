# Convert a fusion VRanges object to left (5') and right (3') GRanges objects

Convert a fusion VRanges object to left (5') and right (3') GRanges
objects

## Usage

``` r
extract_fusion_ranges(vr)
```

## Arguments

- vr:

  A VRanges object with RNA fusions from \`readVcfAsVRanges\`

## Value

A list of the 5' and 3' GRanges objects
